## Installation

<Tabs>
	<TabItem label="Gradle (Groovy)">
		<Steps>
			1. Fügen Sie das Repository in Ihre `build.gradle` hinzu

				```gradle
				repositories {
					...
					maven { url 'https://jitpack.io' }
				}
				```

			2. Fügen Sie die Abhängigkeit in Ihre `build.gradle` hinzu

				```gradle
				dependencies {
					implementation 'net.tcgdex:java-sdk:{Version}'
				}
				```
		</Steps>
	</TabItem>

	<TabItem label="Gradle (Kotlin)">
		<Steps>
			1. Fügen Sie das Repository in Ihre `build.gradle.kts` hinzu

				```gradle
				repositories {
					...
					maven(url = "https://jitpack.io")
				}
				```

			2. Fügen Sie die Abhängigkeit in Ihre `build.gradle.kts` hinzu

				```gradle
				dependencies {
					implementation("net.tcgdex:java-sdk:{Version}")
				}
				```
		</Steps>
	</TabItem>

	<TabItem label="Maven">
		<Steps>
			1. Fügen Sie das **Repository** in Ihre `pom.xml` hinzu

				```xml
				<repositories>
					<repository>
						<id>jitpack.io</id>
						<url>https://jitpack.io</url>
					</repository>
				</repositories>
				```

			2. Fügen Sie die **Abhängigkeit** in Ihre `pom.xml` hinzu

				```xml
				<dependency>
					<groupId>net.tcgdex</groupId>
					<artifactId>java-sdk</artifactId>
					<version>{Version}</version>
				</dependency>
				```
		</Steps>
	</TabItem>
</Tabs>
