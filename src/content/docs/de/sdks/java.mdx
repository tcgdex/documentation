## Installation

<Tabs>
	<TabItem label="Gradle">
			Fügen Sie die Abhängigkeit zu Ihrer `build.gradle` hinzu

				```groovy
				dependencies {
					implementation 'net.tcgdex:java-sdk:{Version}'
				}
				```
	</TabItem>

	<TabItem label="Maven">
			Fügen Sie die **Abhängigkeit** zu Ihrer `pom.xml` hinzu

				```xml
				<dependency>
					<groupId>net.tcgdex</groupId>
					<artifactId>java-sdk</artifactId>
					<version>{Version}</version>
				</dependency>
				```
	</TabItem>
</Tabs>
