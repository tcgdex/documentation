## What's available

<CardGrid stagger>
	<Card title="10 languages and counting" icon="heart">
		The whole API is available in more than **10** languages !

		{/* Note: languages that are commented currently have no cards */}
		**Internationnal Languages** :
			- English, French, Spanish
			- Italian{/* , Portuguese (Portugal) */}, Portuguese (Brazilian), German{/* , Dutch, Polish, Russian */}

		**Asian Languages** :
			- Japanese{/* , Korean */}, Chinese (Traditionnal), Indonesian, Thai{/* , Chinese (Simplified) */}

		**Coming Soon** :
			- Portuguese (Portugal), Dutch, Polish, Russian
			- Korean, Chinese (Simplified)

		*note: Each languages have different completion levels, <a href="https://api.tcgdex.net/status" target="_blank">check the project status here</a>*
	</Card>
	<Card title="a JSON REST API" icon="add-document">
		A [full featured JSON REST API](/rest) fo get any card informations

		- [Cards](/rest/cards) Endpoints
		- [Sets](/rest/sets) Endpoints
		- [Series](/rest/series) Endpoints
		- [Card](/rest/other-fields) fields Endpoints

		Filtering and paginations is also supported for listing operations

		Multiple SDKs are available to help you get started :
		- [Java](/sdks/java)/[Kotlin](/sdks/kotlin)
		- [PHP](/sdks/php)
		- [JavaScript](/sdks/javascript)/[TypeScript](/sdks/typescript)
	</Card>
	<Card title="a GraphQL API" icon="setting">
	The [GraphQL API](/graphql) allows you to make your own requests
		- `Card` and `Cards` to fetch card(s)
		- `Set` and `Sets` to fetch set(s)
		- `Serie` and `Series` to fetch serie(s)

		The queries support filtering and pagination
	</Card>
</CardGrid>
