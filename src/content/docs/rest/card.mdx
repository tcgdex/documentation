## Request
<Tabs syncKey="sdk">
	<TabItem label="PHP">
		```php
		$card = $tcgdex->card->get('swsh3-136');
		```
		_[See how to setup the PHP SDK](/sdks/php)_
	</TabItem>

	<TabItem label="Javascript">
		```javascript
		const card = await tcgdex.card.get('swsh3-136');
		```
		_[See how to setup the Javascript SDK](/sdks/javascript)_

	</TabItem>

	<TabItem label="Typescript">
		```typescript
		const card = await tcgdex.card.getCard('swsh3-136');
		```
		_[See how to setup the Typescript SDK](/sdks/typescript)_

	</TabItem>

	<TabItem label="Java">
		```java
		Card card = tcgdex.fetchCard("swsh3-136");
		```
		_[See how to setup the Java SDK](/sdks/java)_
	</TabItem>

	<TabItem label="Kotlin">
		```kotlin
		val card = tcgdex.fetchCard("swsh3-136")
		```
		_[See how to setup the Kotlin SDK](/sdks/kotlin)_
	</TabItem>

	<TabItem label="cURL">
		```
		https://api.tcgdex.net/v2/en/cards/swsh3-136
		```
		_[See how to prepare your URLs](/rest)_
	</TabItem>
</Tabs>

### 404 Response

The card was not found :( it will return a global error message (not translated)

```json
{
	"error": "Endpoint or id not found"
}
```
