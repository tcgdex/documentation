## Requête
<Tabs syncKey="sdk">
	<TabItem label="PHP">
		```php
		$card = $tcgdex->fetchCard('swsh3-136');
		```
		_[Voir comment configurer le SDK PHP](/sdks/php)_
	</TabItem>

	<TabItem label="Javascript">
		```javascript
		const card = await tcgdex.fetch('cards', 'swsh3-136');
		```
		_[Voir comment configurer le SDK JavaScript](/sdks/javascript)_
	</TabItem>

	<TabItem label="Typescript">
		```typescript
		const card = await tcgdex.fetch('cards', 'swsh3-136');
		```
		_[Voir comment configurer le SDK TypeScript](/sdks/typescript)_
	</TabItem>

	<TabItem label="Java">
		```java
		Card card = tcgdex.fetchCard("swsh3-136");
		```
		_[Voir comment configurer le SDK Java](/sdks/java)_
	</TabItem>

	<TabItem label="Kotlin">
		```kotlin
		val card = tcgdex.fetchCard("swsh3-136")
		```
		_[Voir comment configurer le SDK Kotlin](/sdks/kotlin)_
	</TabItem>

	<TabItem label="cURL">
		```
		https://api.tcgdex.net/v2/fr/cards/swsh3-136
		```
		_[Voir comment préparer vos URLs](/rest)_
	</TabItem>
</Tabs>
