## Requête
<Tabs syncKey="sdk">
	<TabItem label="PHP">
		```php
		$set = $tcgdex->fetchSet('swsh3');
		```
		_[Voir comment configurer le SDK PHP](/sdks/php)_
	</TabItem>
	<TabItem label="Javascript">
		```javascript
		const set = await tcgdex.fetch('sets', 'swsh3');
		```
		_[Voir comment configurer le SDK Javascript](/sdks/javascript)_

	</TabItem>
	<TabItem label="Typescript">
		```typescript
		const set = await tcgdex.fetch('sets', 'swsh3');
		```
		_[Voir comment configurer le SDK Typescript](/sdks/typescript)_

	</TabItem>
	<TabItem label="Java">
		```java
		Set set = tcgdex.fetchSet("swsh3");
		```
		_[Voir comment configurer le SDK Java](/sdks/java)_

	</TabItem>
	<TabItem label="Kotlin">
		```kotlin
		val set = tcgdex.fetchSet("swsh3")
		```
		_[Voir comment configurer le SDK Kotlin](/sdks/kotlin)_
	</TabItem>
	<TabItem label="cURL">
		```
		https://api.tcgdex.net/v2/fr/sets/swsh3
		```
		_[Voir comment préparer vos URLs](/rest)_

	</TabItem>
</Tabs>
