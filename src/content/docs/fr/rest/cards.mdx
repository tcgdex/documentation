## Exemple de Réponse

### Réponse 200

La requête retourne [un tableau d'objets `CardBrief`](/reference/card-brief)

```json
[
	{
		"id": "base4-1",
		"localId": "1",
		"name": "Alakazam",
		"image": "https://assets.tcgdex.net/fr/base/base4/1"
	},
	// ...
	{
		"id": "xy1-1",
		"localId": "1",
		"name": "Florizarre EX",
		"image": "https://assets.tcgdex.net/fr/xy/xy1/1"
	}
]
```
