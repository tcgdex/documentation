## Requête
<Tabs syncKey="sdk">
	<TabItem label="PHP">
		```php
		// récupération du set
		$sets = $tcgdex->getchSets();
		```
		_[Voir comment configurer le SDK PHP](/sdks/php)_

		_Le SDK ne supporte actuellement pas [le filtrage](/rest/filtering-sorting-pagination)_
	</TabItem>

	<TabItem label="Javascript">
		```javascript
		// récupération du set
		const sets = await tcgdex.fetch('sets');
		```
		_[Voir comment configurer le SDK Javascript](/sdks/javascript)_

		_Le SDK ne supporte actuellement pas [le filtrage](/rest/filtering-sorting-pagination)_
	</TabItem>

	<TabItem label="Typescript">
		```typescript
		// récupération du set
		const sets = await tcgdex.fetch('sets');
		```
		_[Voir comment configurer le SDK Typescript](/sdks/typescript)_

		_Le SDK ne supporte actuellement pas [le filtrage](/rest/filtering-sorting-pagination)_
	</TabItem>

	<TabItem label="Java">
		```java
		// récupération du set
		ArrayList<SetResume> sets = tcgdex.getchSets();
		```
		_[Voir comment configurer le SDK Java](/sdks/java)_

		_Le SDK ne supporte actuellement pas [le filtrage](/rest/filtering-sorting-pagination)_
	</TabItem>

	<TabItem label="Kotlin">
		```kotlin
		// Retourne un object set avec toute les informations !
		val sets = tcgdex.getchSets()
		```
		_[Voir comment configurer le SDK Kotlin](/sdks/kotlin)_

		_Le SDK ne supporte actuellement pas [le filtrage](/rest/filtering-sorting-pagination)_
	</TabItem>

	<TabItem label="cURL">
		```
		https://api.tcgdex.net/v2/fr/sets
		```
		_[Voir comment préparer vos URLs](/rest)_

		_[Documentation sur le filtrage](/rest/filtering-sorting-pagination)_
	</TabItem>
</Tabs>
