## Exemple de Réponse

La requête retourne [un tableau d'objets `SerieBrief`](/reference/serie-brief)

```json
[
	{
		"id": "base",
		"name": "Base",
		"logo": "https://assets.tcgdex.net/fr/base/base1/logo"
	},
	// ...
	{
		"id": "xy",
		"name": "XY",
		"logo": "https://assets.tcgdex.net/fr/xy/xyp/logo"
	}
]
```
