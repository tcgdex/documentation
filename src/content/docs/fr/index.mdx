## Ce qui est disponible

<CardGrid stagger>
	<Card title="10 langues et plus" icon="heart">
		L'ensemble de l'API est disponible en plus de **10** langues !

		{/* Remarque : les langues commentées n'ont actuellement pas de cartes */}
		**Langues internationales** :
			- Anglais, Français, Espagnol
			- Italien{/* , Portugais (Portugal) */}, Portugais (Brésilien), Allemand{/* , Néerlandais, Polonais, Russe */}

		**Langues asiatiques** :
			- Japonais{/* , Coréen */}, Chinois (Traditionnel), Indonésien, Thaïlandais{/* , Chinois (Simplifié) */}

		**Bientôt disponible** :
			- Portugais (Portugal), Néerlandais, Polonais, Russe
			- Coréen, Chinois (Simplifié)

		*Remarque : Chaque langue a différents niveaux de complétion, <a href="https://api.tcgdex.net/status" target="_blank">vérifiez l'état du projet ici</a>*
	</Card>

	<Card title="une API REST JSON" icon="add-document">
		Une [API REST JSON complète](/fr/rest) pour obtenir des informations sur n'importe quelle carte

		- Endpoints des cartes
		- Endpoints des ensembles
		- Endpoints des séries
		- Endpoints des champs de cartes

		Le filtrage et la pagination sont également pris en charge pour les opérations de listing

		Plusieurs SDKs sont disponibles pour vous aider à démarrer
	</Card>
	<Card title="une API GraphQL" icon="setting">
	L'[API GraphQL](/fr/graphql) vous permet de faire vos propres requêtes
		- `Card` et `Cards` pour récupérer des carte(s)
		- `Set` et `Sets` pour récupérer des ensemble(s)
		- `Serie` et `Series` pour récupérer des série(s)

		Les requêtes supportent le filtrage et la pagination
	</Card>
</CardGrid>
