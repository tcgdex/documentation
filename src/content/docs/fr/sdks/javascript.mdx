## Installation

<Tabs syncKey='pouet'>
	<TabItem label="NPM">
		```bash
		npm i @tcgdex/sdk
		```
	</TabItem>

	<TabItem label="Bun">
		```bash
		bun i @tcgdex/sdk
		```
	</TabItem>

	<TabItem label="Yarn">
		```bash
		yarn add @tcgdex/sdk
		```
	</TabItem>

	<TabItem label="PNPM">

## Utilisation

<Tabs syncKey='pouet'>
	<TabItem label="Node/Bun">
		```javascript
		// Importer le SDK avec moduleJS
		import TCGdex from '@tcgdex/sdk'
		// OU avec commonJS
		const TCGdex = require('@tcgdex/sdk').default

		// Instancier le SDK
		const tcgdex = new TCGdex('en');

		// entrer dans un contexte async
		(async () => {
			// La carte sera Furret du set Ténèbres Embrasées
			const card = await tcgdex.fetch('cards', 'swsh3-136');
		})();
		```
	</TabItem>

	<TabItem label="Navigateur">
		```html
		<script>
			// Instancier le SDK
			const tcgdex = new TCGdex('en');

			// entrer dans un contexte async
			;(async () => {
				// La carte sera Furret du set Darkness Ablaze
				const card = await tcgdex.fetch('cards', 'swsh3-136');
			})();
		</script>
		```
	</TabItem>
</Tabs>

Vous pouvez consulter les [docs de l'API REST JSON](/rest) et sélectionner l'onglet correct pour voir la commande spécifique à votre SDK
