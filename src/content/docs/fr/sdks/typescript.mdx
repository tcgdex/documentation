## Installation

<Tabs>
	<TabItem label="NPM">
		```bash
		npm i @tcgdex/sdk
		```
	</TabItem>

	<TabItem label="Bun">
		```bash
		bun i @tcgdex/sdk
		```
	</TabItem>

	<TabItem label="Yarn">
		```bash
		yarn add @tcgdex/sdk
		```
	</TabItem>

	<TabItem label="PNPM">
		```bash
		pnpm add @tcgdex/sdk
		```
	</TabItem>
</Tabs>

<Aside title="Utilisation dans le navigateur" type="tip">
TypeScript n'est pas destiné à être exécuté directement dans le navigateur.

Vous voudrez peut-être consulter [le SDK JavaScript](/sdks/javascript) pour ce cas.
</Aside>

## Utilisation

```typescript
// Importer le SDK
import TCGdex from '@tcgdex/sdk'

// Instancier le SDK
const tcgdex = new TCGdex('en');

// entrer dans un contexte async
(async () => {
	// La carte sera Furret du set Darkness Ablaze
	const card = await tcgdex.fetch('cards', 'swsh3-136');
})();
```

Vous pouvez consulter les [docs de l'API REST JSON](/rest) et sélectionner l'onglet correct pour voir la commande spécifique à votre SDK
