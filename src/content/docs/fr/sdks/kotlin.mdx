## Installation

<Tabs>
	<TabItem label="Gradle (Groovy)">
		<Steps>
			1. ajoutez le dépôt dans votre `build.gradle`

				```gradle
				repositories {
					...
					maven { url 'https://jitpack.io' }
				}
				```

			2. Ajoutez la dépendance dans votre `build.gradle`

				```gradle
				dependencies {
					implementation 'net.tcgdex:java-sdk:{Version}'
				}
				```
		</Steps>
	</TabItem>

	<TabItem label="Gradle (Kotlin)">
		<Steps>
			1. ajoutez le dépôt dans votre `build.gradle.kts`

				```gradle
				repositories {
					...
					maven(url = "https://jitpack.io")
				}
				```

			2. Ajoutez la dépendance dans votre `build.gradle.kts`

				```gradle
				dependencies {
					implementation("net.tcgdex:java-sdk:{Version}")
				}
				```
		</Steps>
	</TabItem>

	<TabItem label="Maven">
		<Steps>
			1. ajoutez le **dépôt** dans votre `pom.xml`

				```xml
				<repositories>
					<repository>
						<id>jitpack.io</id>
						<url>https://jitpack.io</url>
					</repository>
				</repositories>
				```

			2. Ajoutez la **dépendance** dans votre `pom.xml`

				```xml
				<dependency>
					<groupId>net.tcgdex</groupId>
					<artifactId>java-sdk</artifactId>
					<version>{Version}</version>
				</dependency>
				```
		</Steps>
	</TabItem>
</Tabs>
