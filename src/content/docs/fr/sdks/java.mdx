## Installation

<Tabs>
	<TabItem label="Gradle">
			Ajoutez la dépendance dans votre `build.gradle`

				```groovy
				dependencies {
					implementation 'net.tcgdex:java-sdk:{Version}'
				}
				```
	</TabItem>

	<TabItem label="Maven">
			Ajoutez la **dépendance** dans votre `pom.xml`

				```xml
				<dependency>
					<groupId>net.tcgdex</groupId>
					<artifactId>java-sdk</artifactId>
					<version>{Version}</version>
				</dependency>
				```
	</TabItem>
</Tabs>
