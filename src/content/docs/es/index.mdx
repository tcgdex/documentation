<CardGrid stagger>

		¡La API completa está disponible en más de **10** idiomas!

	<Card title="una API REST en JSON" icon="add-document">
		Una [API REST en JSON completa](/es/rest) para obtener cualquier información sobre las cartas

		- Endpoints de cartas
		- Endpoints de conjuntos
		- Endpoints de series
		- Endpoints de campos de cartas

		El filtrado y la paginación también son compatibles para las operaciones de listado

		Hay varios SDKs disponibles para ayudarte a comenzar
	</Card>
	<Card title="una API GraphQL" icon="setting">
	La [API GraphQL](/es/graphql) te permite hacer tus propias solicitudes
		- `Card` y `Cards` para obtener carta(s)
		- `Set` y `Sets` para obtener conjunto(s)
		- `Serie` y `Series` para obtener serie(s)

		Las consultas admiten filtrado y paginación
	</Card>
