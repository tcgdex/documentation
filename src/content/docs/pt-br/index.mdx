<CardGrid stagger>

	<Card title="uma API REST JSON" icon="add-document">
		Uma [API REST JSON completa](/pt-br/rest) para obter informações sobre qualquer carta

		- Endpoints de cartas
		- Endpoints de conjuntos
		- Endpoints de séries
		- Endpoints de campos de cartas

		A filtragem e a paginação também são suportadas para operações de listagem

		Vários SDKs estão disponíveis para ajudar você a começar
	</Card>
	<Card title="uma API GraphQL" icon="setting">
	A [API GraphQL](/pt-br/graphql) permite que você faça suas próprias solicitações
		- `Card` e `Cards` para buscar carta(s)
		- `Set` e `Sets` para buscar conjunto(s)
		- `Serie` e `Series` para buscar série(s)

		As consultas suportam filtragem e paginação
	</Card>
