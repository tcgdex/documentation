## Installation

<Tabs>
	<TabItem label="Gradle (Groovy)">
			Add the dependency in your `build.gradle`

				```groovy
				dependencies {
					implementation 'net.tcgdex:sdk:{Version}'
				}
				```
	</TabItem>

	<TabItem label="Gradle (Kotlin)">
			Add the dependency in your `build.gradle.kts`

				```groovy
				dependencies {
					implementation("net.tcgdex:sdk:{Version}")
				}
				```
	</TabItem>

	<TabItem label="Maven">
			Add the **depedency** in your `pom.xml`

				```xml
				<dependency>
					<groupId>net.tcgdex</groupId>
					<artifactId>sdk</artifactId>
					<version>{Version}</version>
				</dependency>
				```
	</TabItem>
</Tabs>
