## Installation

<Tabs syncKey='pouet'>

		```bash
		npm i @tcgdex/sdk
		```
	</TabItem>

	<TabItem label="Bun">

## Usage

<Tabs syncKey='pouet'>
	<TabItem label="Node/Bun">
		```javascript
		// Import the SDK in moduleJS
		import TCGdex from '@tcgdex/sdk'
		// OR import the SDK in commonJS
		const TCGdex = require('@tcgdex/sdk').default

		// Instantiate the SDK
		const tcgdex = new TCGdex('en');

		// go into an async context
		(async () => {
			// Card will be Furret from the Darkness Ablaze Set
			const card = await tcgdex.card.get('swsh3-136');
		})();
		```
	</TabItem>

	<TabItem label="Browser">
		```html
		<script>
			// Instantiate the SDK
			const tcgdex = new TCGdex('en');

			// go into an async context
			;(async () => {
				// Card will be Furret from the Darkness Ablaze Set
				const card = await tcgdex.card.get('swsh3-136');
			})();
		</script>
		```
	</TabItem>
</Tabs>

You can check in [the JSON REST API docs](/rest) and select the correct tab to see the command specific for your SDK
