## Installation

<Tabs>
	<TabItem label="Gradle">
		<Steps>
			1. add the repository in your `build.gradle`

				```gradle
				repositories {
					...
					maven { url 'https://jitpack.io' }
				}
				```

			2. Add the dependency in your `build.gradle`

				```gradle
				dependencies {
					implementation 'net.tcgdex:java-sdk:{Version}'
				}
				```
		</Steps>
	</TabItem>

	<TabItem label="Maven">
		<Steps>
			1. add the **repository** in your `pom.xml`

				```xml
				<repositories>
					<repository>
						<id>jitpack.io</id>
						<url>https://jitpack.io</url>
					</repository>
				</repositories>
				```

			2. Add the **depedency** in your `pom.xml`

				```xml
				<dependency>
					<groupId>net.tcgdex</groupId>
					<artifactId>java-sdk</artifactId>
					<version>{Version}</version>
				</dependency>
				```
		</Steps>
	</TabItem>
</Tabs>
